Clone repo

```
$ git clone https://github.com/megadrive/OBS-Scene-Scripter
```

install deps

```
$ cd OBS-Scene-Scripter
$ npm i
```

change `src/index.ts` to refer to a name of an actual scene/source item then compile

```
$ tsc
```

and run 

```
$ node dist/index.js
```

watch our obs window wow
